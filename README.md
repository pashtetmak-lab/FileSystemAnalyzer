# FileSystemAnalyzer

## 1. Описание проекта
FileSystemAnalyzer — консольное приложение на C++, которое сканирует файловую систему, строит дерево каталогов и файлов, вычисляет размеры директорий и выводит результат в читаемом виде.

## 4. Краткое описание
Входные данные:
- путь к файлу или директории;
- опциональный аргумент `--depth N` для ограничения глубины обхода.

Результат работы:
- дерево файлов и директорий;
- размер файлов в байтах;
- суммарный размер директорий, посчитанный по дочерним узлам.

## 5. Использованные современного C++
- `std::unique_ptr`:
  используется в `FsNode` для владения дочерними узлами дерева без ручного управления памятью.
- `std::optional`:
  используется в `FileSystemScanner::ScanConfig` для хранения необязательного ограничения глубины обхода.
- `std::variant`:
  используется в `FsNode` для хранения одного из двух состояний узла: файл (`FileInfo`) или директория (`DirInfo`).
- RAII:
  проект не использует `new/delete` вручную; память, файловые пути и контейнеры управляются средствами стандартной библиотеки.
- Исключения:
  на верхнем уровне в `Application` используется `try/catch`, чтобы перехватывать ошибки и выводить сообщение пользователю.
- `std::filesystem`:
  применяется для обхода файловой системы, проверки существования путей, определения типа объекта и получения размеров файлов.

## 6. Сборка проекта
Требования:
- компилятор с поддержкой C++23
- CMake 4.2.3
- стандартная библиотека C++ с поддержкой `std::filesystem`


Запуск с ограничением глубины:

./build/FileSystemAnalyzer /path/to/folder --depth 2

## 7. Пример работы
