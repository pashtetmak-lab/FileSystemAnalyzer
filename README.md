# FileSystemAnalyzer

## 1. Название и цель проекта
FileSystemAnalyzer — консольное приложение на C++, которое сканирует файловую систему, строит дерево каталогов и файлов, вычисляет размеры директорий и выводит результат в читаемом виде.
